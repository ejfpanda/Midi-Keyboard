# Midi-Keyboard

## Overview

It has 20 keys, 2 rotary encoders, a digital-to-analog converter (I2S DAC), an sd card, LCD and uses a raspberry pi pico. You can use the rotary encoders to shift octaves and also change midi channels. The microcontroller is used to process user input into this midi keyboard.
